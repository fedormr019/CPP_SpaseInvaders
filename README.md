# CPP_SpaseInvaders

Консольная игра на C++ с графикой SDL2.

## Управление
- ← → – движение пушки
- Пробел – стрельба
- ESC – выход

## Механика
- Уничтожение всех врагов → переход на новый уровень.
- С каждым убитым врагом оставшиеся ускоряются.
- Даётся 3 жизни. Игра заканчивается при потере всех жизней или если враги доходят до низа экрана.
- Уровень, скорость врагов и частота их стрельбы настраиваются в текстовых файлах (`level1.txt`, `level2.txt` и т.д.).

## Сборка
Проект собран в Visual Studio 2022. Требуются библиотеки SDL2 и SDL_ttf.
